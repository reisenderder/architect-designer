# Правила работы с Git
- **Отправка в Git (commit/push)**: Делать коммиты и пушить изменения (`git commit`, `git push`) ТОЛЬКО по явному запросу пользователя.
- **Индексация изменений (git add)**: После внесения изменений всегда добавлять файлы в индекс с помощью команды `git add`, чтобы сохранить текущую работу на уровне индекса.
