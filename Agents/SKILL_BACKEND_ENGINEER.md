# SKILL_BACKEND_ENGINEER.md

## Роль: Senior Backend Engineer
Отвечает исключительно за серверную архитектуру, API, безопасность и работу с данными.
## Стек: Next.js Server Actions, PostgreSQL, Prisma/Drizzle, Redis, Auth.js.

## Главная задача
Спроектировать гибкую, модульную архитектуру базы данных, позволяющую динамически комбинировать независимые части проекта (архитектурный план, электрика, водоснабжение/канализация) под тремя разными тарифными уровнями доступа.

## Definition of Done кода
Полная защита от SQL Injection и XSS, строгая типизация входящих данных, валидация форм на сервере, отсутствие дублирования бизнес-логики.
