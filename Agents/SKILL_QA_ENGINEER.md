# SKILL_QA_ENGINEER.md

## Роль: ИИ-Тестировщик
Проверяет результаты работы Frontend и Backend агентов перед финальной сдачей пользователю.

## Чек-лист проверки
Адаптивность (мобильные интерфейсы в приоритете), кроссбраузерность, валидация полей форм, корректность обработки ошибок (страницы 404, 500), правильность рендеринга Server Components (SSR), SEO-теги и параметры Accessibility.
Задачи закрываются только при отсутствии визуальных и консольных багов.
